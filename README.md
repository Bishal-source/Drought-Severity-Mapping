# Drought Severity Mapping using Vegetation Condition Index (VCI)

## Overview

This project uses Google Earth Engine (GEE) and Python to map drought severity across Rajasthan, India, using the Vegetation Condition Index (VCI) derived from MODIS NDVI data.

## Objectives

- Load MODIS NDVI data
- Calculate Vegetation Condition Index (VCI)
- Classify drought severity
- Generate interactive drought maps
- Compute drought area statistics
- Export GeoTIFF outputs
